1. Download phoronix
https://www.phoronix-test-suite.com/?k=downloads

2. Install
sudo dpkg -i phoronix-test-NNN.deb

3. List tests
phoronix-test-suite list-tests

4. Install unigine-heaven
phoronix-test-suite install unigine-heaven

5. Run
phoronix-test-suite run unigine-heaven

-----

# Test With Steam

1. Install PTS supported game in Steam

2. Leave Steam open

3. Install the test with PTS
phoronix-test-suite install supertuxkart

4. run test
phoronix-test-suite run supertuxkart
